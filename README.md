# smart-budget-tracker
A mobile app to help users track expenses, manage budgets, and analyze spending trends.

## 🚀 Features

### Core Features
- 🔑 **User Authentication** – Secure login and signup (Firebase Auth: Email/Google)  
- 💰 **Expense Tracking** – Add, edit, and delete expenses with categories and dates  
- 🎯 **Budget Management** – Set monthly or weekly budget goals  
- 📊 **Analytics Dashboard** – Interactive charts to monitor spending trends  
- 📶 **Offline Support** – Store data locally and sync when back online  

### Stretch Features (If Time Allows)
- 🔔 **Push Notifications** – Alerts when budget limits are exceeded  
- 🤖 **AI Insights** – Smart recommendations on spending habits  
- 📤 **Data Export** – Generate CSV or PDF reports  


## 🛠 Tech Stack
- **Frontend:** React Native (Expo)  
- **Navigation:** React Navigation  
- **UI Components:** React Native Paper  
- **Charts & Visualization:** react-native-chart-kit  
- **Backend & Database:** Firebase (Firestore + Auth)  
- **Deployment:** Expo Go / EAS Build
